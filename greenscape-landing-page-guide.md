# GreenScape Lawn & Garden - Complete Landing Page Guide

## 1. Recommended Free WordPress Theme

**Theme: Astra (Free Version)**
- Lightweight and fast
- Excellent Elementor compatibility
- Mobile-responsive out of the box
- Clean, professional design
- Easy customization options

**Alternative Options:**
- OceanWP (Free)
- GeneratePress (Free)
- Kadence (Free)

## 2. Color Palette

**Primary Colors:**
- Forest Green: `#2D5016` (Primary brand color - headers, CTAs)
- Fresh Green: `#6B9F3E` (Accent - hover states, highlights)
- Earth Brown: `#8B6F47` (Secondary accent - grounding element)

**Neutral Colors:**
- Cream White: `#F9F7F4` (Background sections)
- Pure White: `#FFFFFF` (Main background)
- Charcoal: `#333333` (Body text)
- Light Gray: `#E8E8E8` (Borders, dividers)

**Accent:**
- Sky Blue: `#87CEEB` (Optional - for water features/irrigation services)

## 3. Typography

**Headings:**
- Font: Montserrat (Google Font)
- Weight: 700 (Bold) for H1, 600 (Semi-Bold) for H2/H3
- Style: Clean, modern, professional

**Body Text:**
- Font: Open Sans (Google Font)
- Weight: 400 (Regular), 600 (Semi-Bold for emphasis)
- Line Height: 1.6 for readability

**Font Sizes:**
- H1: 48px (desktop) / 32px (mobile)
- H2: 36px (desktop) / 28px (mobile)
- H3: 24px (desktop) / 20px (mobile)
- Body: 16px
- Small text: 14px

## 5. Step-by-Step Elementor Build Instructions

### Initial Setup:

1. **Install Required Plugins:**
   - Install Elementor (Free version)
   - Install WPForms Lite OR Contact Form 7
   - Install Astra theme

2. **Theme Configuration:**
   - Go to Appearance > Customize
   - Set colors: Primary color to #2D5016
   - Set fonts: Headings to Montserrat, Body to Open Sans
   - Disable default header/footer for landing page (optional)

3. **Create New Page:**
   - Pages > Add New
   - Title: "Home" or "Landing Page"
   - Click "Edit with Elementor"

### Building Each Section:

**Hero Section:**
1. Add new section (full-width)
2. Section Settings > Style > Background Type: Image
3. Upload hero image (recommended: 1920x1080px)
4. Background Overlay: Enable, Color: Black, Opacity: 40%
5. Add Heading widget: Type H1 text, center align, white color
6. Add Text Editor widget: Supporting text, center align, white color
7. Add Button widget: Primary button style, link to contact form (#contact)
8. Column Settings: Vertical Align: Middle, Content Width: 800px

**About Us Section:**
1. Add new section (boxed width)
2. Section background: #F9F7F4
3. Add two columns (50/50 split)
4. Left column: Add Image widget, upload team/equipment photo
5. Right column: Add Heading (H2), Text Editor, Icon List widgets
6. Mobile: Stack columns, image on top

**Services Section:**
1. Add new section (boxed width)
2. Add four columns (25% each) or three columns (33% each)
3. Each column: Add Icon Box widget
4. Icon Box settings: Icon at top, center aligned
5. Add icon, H3 heading, description text, "Learn More" link
6. Style: Add border, padding, hover effects
7. Mobile: Stack columns vertically

**Testimonials Section:**
1. Add new section (boxed width)
2. Section background: #F9F7F4
3. Add three columns (33% each)
4. Each column: Add Testimonial widget (or Icon Box styled as testimonial)
5. Include star rating, quote text, customer name
6. Style: White background cards with shadow
7. Mobile: Consider using slider/carousel for better UX

**Contact Section:**
1. Add new section (boxed width)
2. Add two columns (40/60 split)
3. Left column: Add Heading (H2), Text Editor with contact info, Icon List
4. Right column: Add Shortcode widget
5. Insert form shortcode: `[wpforms id="123"]` or `[contact-form-7 id="123"]`
6. Style form fields to match color palette

**Footer CTA:**
1. Add new section (full-width)
2. Section background: #2D5016
3. Add Heading (H2), Text Editor, Button widgets
4. All text: White color
5. Button: Inverted style (white background, green text)

### General Elementor Tips:

**Spacing:**
- Section padding: 80px top/bottom (desktop), 40px (mobile)
- Column gap: 30px
- Widget spacing: 20px between elements

**Responsive Design:**
- Always preview in mobile/tablet views
- Adjust font sizes for mobile (use responsive controls)
- Stack columns on mobile
- Reduce padding on mobile
- Ensure buttons are thumb-friendly (min 44px height)

**Animations (Optional):**
- Entrance animations: Fade In, Fade In Up
- Duration: 600-800ms
- Don't overuse - keep it subtle

## 6. Contact Form Setup

### WPForms Configuration:

1. Go to WPForms > Add New
2. Choose "Simple Contact Form" template
3. Customize fields as listed above
4. Form Settings:
   - Form Name: "GreenScape Quote Request"
   - Submit Button Text: "Request Free Quote"
5. Notifications:
   - Send to: your-email@greenscapelawn.com
   - Subject: "New Quote Request from {field_id='1'}"
6. Confirmations:
   - Type: Message
   - Message: "Thank you! We'll contact you within 24 hours with your free quote."
7. Copy shortcode and paste into Elementor Shortcode widget

## 7. Mobile Responsiveness Tips

**Critical Checks:**
- Test on actual devices or Chrome DevTools
- Ensure text is readable (min 16px body text)
- Buttons are easily tappable (min 44x44px)
- Images scale properly and don't overflow
- Forms are easy to fill on mobile
- No horizontal scrolling
- Navigation is accessible (hamburger menu)

**Elementor Mobile Settings:**
- Use responsive controls (desktop/tablet/mobile icons)
- Hide decorative elements on mobile if needed
- Adjust column widths: Desktop (25/25/25/25) → Mobile (100%)
- Reduce section padding on mobile
- Stack columns vertically
- Increase line height for better mobile readability

**Mobile-Specific Adjustments:**
- Hero: Reduce height to 500px, smaller heading size
- Services: Single column layout
- Testimonials: Slider/carousel instead of grid
- Contact: Stack form above contact info
- Reduce image sizes for faster loading

## 8. SEO Best Practices

**Page Settings:**
- Page Title: "Professional Lawn Care & Landscaping Services | GreenScape Lawn & Garden"
- Meta Description: "Transform your outdoor space with GreenScape's professional lawn care, garden design, and landscaping services in [Your City]. Free quotes available. Call (555) 123-4567."
- URL Slug: "/" (homepage) or "/landscaping-services"

**On-Page SEO:**
- Use only ONE H1 tag (in hero section)
- Proper heading hierarchy: H1 > H2 > H3
- Include location keywords naturally in copy
- Add alt text to all images: "Professional lawn mowing service in [City]"
- Internal linking if you add more pages later
- Fast loading speed (optimize images, use WebP format)

**Image Optimization:**
- Compress all images before upload (use TinyPNG or similar)
- Recommended sizes:
  - Hero image: 1920x1080px, under 200KB
  - Service icons: SVG format (scalable)
  - About image: 800x600px, under 150KB
  - Testimonial photos: 150x150px, under 50KB

**Schema Markup (Advanced):**
- Install Rank Math or Yoast SEO plugin
- Add LocalBusiness schema
- Include business hours, phone, address
- Add Service schema for each service type
